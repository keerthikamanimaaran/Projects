# Job Application Tracker

## 📌 Project Overview

Job Application Tracker is a command-line based backend application developed using Python, SQLite, SQL, and Pandas.

The application helps users manage and track their job applications efficiently by supporting CRUD operations, analytics, search functionality, and CSV export.

---

# 🚀 Features

## ✅ Application Management
- Add job applications
- View stored applications
- Update application status
- Delete applications

## ✅ Search Functionality
- Search applications using:
  - Company name
  - Role
  - Status

## ✅ Analytics
- Total applications count
- Status summary
- Success rate calculation
- Most applied company

## ✅ CSV Export
- Export application data into CSV format

## ✅ Validation & Exception Handling
- Empty field validation
- Date format validation
- Invalid ID handling
- Exception handling using try-except
